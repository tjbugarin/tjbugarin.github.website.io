Algorithms

At the beginning of this class, we learned about the usage of algorithms in the world of coding. An algorithm is basically a set of rules that one follws in order to produce anything like a hidden message, a code, or in our case, a password. Our very first major assignment that I felt really introduced us into challenging computer science work was our password generator assignment. It involved creating steps (almost like an algorithm) so create unique passwords.

Password Generator Steps

Step 1: (If websites name begins with letters) First letter lowercase, Second letter uppercase of websites name Step 2: Name of site is alphabetically translated into numbers (ex:A=1, B=2, C=3, etc.) Step 3: Color of sites brightest color in all uppercase Step 4: Take out all vowels in sites name Step 5: Last character is websites name lowercase followed by a underscore.(if symbol or number just state it as is) Step 6: how many characters are in the websites titles name Step 7: The fourth character of the websites name is capitalized (if symbol or number just state it as is) Step 8: first and lasts letter capitalized of the website (if symbol or number just state it as is)
